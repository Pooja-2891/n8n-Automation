# AI-Powered Customer Feedback Automation (n8n)

This project automates customer feedback analysis using n8n.

## Features
- AI sentiment analysis using Gemini
- Automatic classification:
  - Complaints
  - Compliments
  - Feature Requests
- Stores feedback in Airtable
- Sends alerts via Slack and Gmail

## Technologies
- n8n
- Google Gemini
- Airtable
- Slack
- Gmail

## Workflow
See workflow.json to import into n8n.
